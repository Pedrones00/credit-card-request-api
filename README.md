# Tech Challenge

Neste Tech Challenge fomos desafiados a criar um sistema que gerencie cartões, clientes e contratos.

## Tecnologias utilizadas

Para desenvolver esse sistema utilizamos as seguintes tecnologias:

- NodeJS
- MySQL

## Como instalar

```bash
    npm install
```

## Como executar

```bash
    npm run dev
```
